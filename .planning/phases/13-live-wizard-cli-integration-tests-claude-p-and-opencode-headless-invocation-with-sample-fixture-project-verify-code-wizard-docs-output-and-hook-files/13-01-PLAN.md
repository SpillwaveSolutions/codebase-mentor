**Module docstring:**
```
Live integration tests -- real LLM calls via claude -p and opencode run.

Run with: pytest -m slow tests/test_wizard_live.py
Skip in fast CI: pytest -m 'not slow'

Requires:
  - claude CLI installed and authenticated (OAuth or ANTHROPIC_API_KEY)
  - opencode CLI installed and authenticated
  - codebase-wizard plugin installed globally via ai-codebase-mentor
```

**Imports:** os, shutil, subprocess, pathlib.Path, pytest

**Constants:**
- `FIXTURE_PROJECT = Path(__file__).parent / "fixtures" / "sample-wizard-project"`

**Auth probe functions (module-level, evaluated once at import):**

`_probe_claude()` -> bool:
- Return False if `shutil.which("claude") is None`
- Run `subprocess.run(["claude", "-p", "OK", "--dangerously-skip-permissions", "--max-budget-usd", "0.01"], capture_output=True, text=True, timeout=30)`
- Return `r.returncode == 0`
- Wrap in try/except, return False on any exception

`_probe_opencode()` -> bool:
- Return False if `shutil.which("opencode") is None`
- Run `subprocess.run(["opencode", "run", "OK"], capture_output=True, text=True, timeout=30)`
- Return `r.returncode == 0`
- Wrap in try/except, return False on any exception

Module-level variables:
```python
CLAUDE_AVAILABLE = _probe_claude()
OPENCODE_AVAILABLE = _probe_opencode()

skip_no_claude = pytest.mark.skipif(not CLAUDE_AVAILABLE, reason="claude not available or not authenticated")
skip_no_opencode = pytest.mark.skipif(not OPENCODE_AVAILABLE, reason="opencode not available or not authenticated")
```

**Fixtures:**

`wizard_project(tmp_path)`:
- Copy FIXTURE_PROJECT to `tmp_path / "project"` via `shutil.copytree`
- Create `(project / ".code-wizard" / "sessions").mkdir(parents=True)`
- Create `(project / ".code-wizard" / "docs").mkdir(parents=True)`
- Write `(project / ".code-wizard" / "config.json")` with: `{"version": 1, "resolved_storage": ".code-wizard", "created": "2026-03-25T00:00:00Z"}`
- Return `project`

**Helpers:**

`run_claude(prompt: str, cwd: Path, budget: float = 0.50, timeout: int = 120) -> subprocess.CompletedProcess`:
- `subprocess.run(["claude", "-p", prompt, "--dangerously-skip-permissions", "--max-budget-usd", str(budget)], cwd=str(cwd), capture_output=True, text=True, timeout=timeout)`

`run_opencode(message: str, cwd: Path, timeout: int = 120) -> subprocess.CompletedProcess`:
- `subprocess.run(["opencode", "run", message, "--dir", str(cwd)], capture_output=True, text=True, timeout=timeout)`

CRITICAL: Do NOT use `--command` flag with opencode (it hangs). Do NOT use slash commands like `/codebase-wizard` with claude -p (returns "Unknown skill").

**Tests (each decorated with @pytest.mark.slow):**

   - Assert `.code-wizard/docs/claude-test-001/SESSION-TRANSCRIPT.md` exists
   - Assert content length > 200 bytes
   - Assert at least one line starts with "# " (has a heading)

2. `test_claude_describe_creates_codebase_md(wizard_project)` -- @skip_no_claude
   - Prompt: "Run the codebase wizard in describe mode. Write SESSION-TRANSCRIPT.md and CODEBASE.md to .code-wizard/docs/claude-test-002/. Proceed without questions."
   - Assert returncode == 0
   - Assert `.code-wizard/docs/claude-test-002/SESSION-TRANSCRIPT.md` exists
   - Assert content length > 100 bytes

3. `test_opencode_describe_creates_transcript(wizard_project)` -- @skip_no_opencode
   - Message: "Run the codebase wizard in describe mode on this project. Write SESSION-TRANSCRIPT.md to .code-wizard/docs/opencode-test-001/. Proceed without asking questions."
   - Assert returncode == 0 with stderr in error message
   - Assert `.code-wizard/docs/opencode-test-001/SESSION-TRANSCRIPT.md` exists
   - Assert content length > 200 bytes

4. `test_claude_skip_when_no_auth()` -- NO skip decorator

   - Assert `isinstance(CLAUDE_AVAILABLE, bool)` (probe ran without crashing)
   - If not CLAUDE_AVAILABLE: `pytest.skip("claude not available -- skip logic confirmed working")`

Do NOT assert on exact heading text (LLM output is non-deterministic). Assert only on: file existence, minimum byte count (> 200), presence of at least one `# ` heading line.
