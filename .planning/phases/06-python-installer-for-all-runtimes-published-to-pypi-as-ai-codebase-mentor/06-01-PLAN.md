from docs/superpowers/specs/2026-03-20-codebase-wizard-multi-runtime-design.md Section 2:

```python
class RuntimeInstaller:
    def install(self, source: Path, target: str = "global") -> None:
        """
        source: path to bundled plugin dir (ai_codebase_mentor/plugin/)
        target: "global" (~/.{runtime}/plugins/) or "project" (./plugins/)
        Raises: RuntimeError if install fails.
        Idempotent: calling install when already installed updates in place.
        """

    def uninstall(self, target: str = "global") -> None:
        """
        Removes all files written by install().
        No-op if not installed (does not raise).
        """

    def status(self) -> dict:
        """
        Returns: {"installed": bool, "location": str | None, "version": str | None}
        """
```

Error contract:
- install() raises RuntimeError with human-readable message on failure
- uninstall() is always safe — no-op if not installed, never raises
- All converters are idempotent for install (update in place if already installed)

3. Create ai_codebase_mentor/converters/base.py implementing the RuntimeInstaller ABC
   exactly as specified in the design spec (Section 2). Use abc.ABC and
   abc.abstractmethod so subclasses are forced to implement all three methods.
   Include a helper property `plugin_source` that returns the path to the bundled
   plugin directory (importlib.resources or __file__-relative path).

   Full implementation:
   ```python
   """Base class for runtime installers."""

   import abc
   from pathlib import Path


   class RuntimeInstaller(abc.ABC):
       """Abstract base for all runtime converters.

       Subclasses implement install/uninstall/status for a specific AI runtime
       (Claude Code, OpenCode, Codex, Gemini, etc.).
       """

       @property
       def plugin_source(self) -> Path:
           """Return path to the bundled plugin directory."""
           return Path(__file__).parent.parent / "plugin"

       @abc.abstractmethod
       def install(self, source: Path, target: str = "global") -> None:
           """Install plugin to the target runtime.

           Args:
               source: Path to bundled plugin dir (ai_codebase_mentor/plugin/).
               target: "global" (~/.{runtime}/plugins/) or "project" (./plugins/).

           Raises:
               RuntimeError: If install fails. Message must be human-readable.

           Note: Idempotent — calling when already installed updates in place.
           """

       @abc.abstractmethod
       def uninstall(self, target: str = "global") -> None:
           """Remove all files written by install().

           Args:
               target: "global" or "project" (must match original install target).

           Note: No-op if not installed. Never raises.
           """

       @abc.abstractmethod
       def status(self) -> dict:
           """Return install status.

           Returns:
               dict with keys:
                   installed (bool): Whether plugin is currently installed.
                   location (str | None): Absolute path to install dir, or None.
                   version (str | None): Installed version string, or None.
           """
   ```
