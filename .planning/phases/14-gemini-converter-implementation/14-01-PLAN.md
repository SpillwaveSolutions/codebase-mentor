class RuntimeInstaller(ABC):
    @property
    def plugin_source(self) -> Path:
        return Path(__file__).parent.parent / "plugin"

    @abstractmethod
    def install(self, source: Path, target: str = "global") -> None: ...

    @abstractmethod
    def uninstall(self, target: str = "global") -> None: ...

    @abstractmethod
    def status(self) -> dict: ...

def _read_version(plugin_dir: Path) -> str | None: ...

    Helper functions:
    1. `_convert_gemini_tool_name(tool: str) -> str | None` — strip path scope with `tool.split("(")[0]`, return None for `mcp__*`, `Task`, `Agent`, otherwise map via GEMINI_TOOL_MAP or fallback to `.lower()`
    2. `_rewrite_paths(content: str) -> str` — apply PATH_REWRITES_GEMINI in order (longer match `$HOME/.claude` first)
    3. `_escape_template_vars(content: str) -> str` — `re.sub(r'\$\{(\w+)\}', r'$\1', content)`

    5. `_convert_agent_frontmatter(content: str) -> str` — line-by-line YAML parser that:
       - Splits on `---` markers to isolate frontmatter from body
       - Iterates frontmatter lines: detects top-level keys, tracks `in_allowed_tools` and `in_block_scalar` state
       - Strips `color:` lines entirely (unlike OpenCode which converts to hex)
       - Strips `skills:` lines entirely (Gemini validator rejects unknown fields)
       - KEEPS `name:` lines (unlike OpenCode which strips them)
       - Collects tool names from `allowed_tools:` block, maps via `_convert_gemini_tool_name()`, excludes None results
       - Deduplicates tools (scoped Write and Edit map to same name)
       - Outputs `tools:` as YAML array format: `tools:\n  - read_file\n  - run_shell_command`
       - Applies content transforms (`_escape_template_vars`, `_strip_sub_tags`, `_rewrite_paths`) to BODY only, not frontmatter
       - Reassembles: `---\n{frontmatter}\n---\n{transformed_body}`
    6. `_convert_command_to_toml(content: str) -> str` — extracts description from YAML frontmatter, applies all content transforms to body, outputs `description = {json.dumps(desc)}\nprompt = {json.dumps(body)}\n`

    Class `GeminiInstaller(RuntimeInstaller)`:
    - `_resolve_dest(self, target: str) -> Path`:
      - If target == "global": check `os.environ.get("GEMINI_CONFIG_DIR")` first, if set return `Path(config_dir).expanduser() / "codebase-wizard"`, else return `Path.home() / ".gemini" / "codebase-wizard"`
      - If target == "project": return `Path.cwd() / ".gemini" / "codebase-wizard"`
    - `install(self, source: Path, target: str = "global") -> None`:
      - Raise RuntimeError if source does not exist
      - resolve dest, rmtree if exists, mkdir parents
      - Convert agents: source/"agents"/*.md -> dest/"agent"/*.md via `_convert_agent_frontmatter` then `_rewrite_paths`
      - Convert commands: source/"commands"/*.md -> dest/"command"/*.toml via `_convert_command_to_toml`
      - Copy skills: `shutil.copytree(source/"skills", dest/"skill")`
      - Copy .claude-plugin: `shutil.copytree(source/".claude-plugin", dest/".claude-plugin")`
      - Wrap in try/except OSError -> RuntimeError
    - `uninstall(self, target: str = "global") -> None`:
      - resolve dest, if exists rmtree, else return (no-op)
    - `status(self) -> dict`:
      - Check global path first (with GEMINI_CONFIG_DIR override), then project path
      - Return `{"installed": True/False, "location": str/None, "version": str/None}`
