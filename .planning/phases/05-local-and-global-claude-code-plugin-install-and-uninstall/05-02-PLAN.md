ClaudeInstaller class in claude.py:

- Imports: `shutil`, `pathlib.Path`, `from .base import RuntimeInstaller, _read_version`
- GLOBAL_DEST = Path.home() / ".claude" / "plugins" / "codebase-wizard"
- PROJECT_DEST = Path("plugins") / "codebase-wizard"  (relative to cwd)

install(self, source: Path, target: str = "global") -> None:
  - Validate source exists; raise RuntimeError("Plugin source directory not found: {source}") if not
  - Resolve destination: GLOBAL_DEST if target == "global", else Path.cwd() / "plugins" / "codebase-wizard"
  - If destination exists, remove it first (shutil.rmtree) to ensure idempotency via fresh copy
  - shutil.copytree(source, destination) — copies entire plugin tree
  - If any OSError occurs, raise RuntimeError("Failed to install to {destination}: {e}")

uninstall(self, target: str = "global") -> None:
  - Resolve destination (same logic as install)
  - If destination does not exist: return silently (no-op)
  - shutil.rmtree(destination)
  - If OSError: raise RuntimeError("Failed to uninstall from {destination}: {e}")

status(self) -> dict:
  - Check GLOBAL_DEST first. If it exists:
    - Return {"installed": True, "location": str(GLOBAL_DEST), "version": _read_version(GLOBAL_DEST)}
  - Check PROJECT_DEST (Path.cwd() / "plugins" / "codebase-wizard"). If it exists:
    - Return {"installed": True, "location": str(PROJECT_DEST.resolve()), "version": _read_version(PROJECT_DEST)}
  - Otherwise: return {"installed": False, "location": None, "version": None}
